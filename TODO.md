## Баги

### 1. Отображение таблиц с ID (extra_verbose)
**Проблема:** При включении опции «Показывать ID в [скобках]» таблицы отображаются некорректно — большинство колонок пустые.

**Причина:** Функции отрисовки используют жёстко заданные имена колонок, но при `extra_verbose=True` имена колонок другие (например, `Дата [duedate]` вместо `Дата`).

**Решение:** Рефакторинг функций отрисовки для динамического определения колонок.

## Оптимизация

### 2. Дублирование запросов к Jira API ✅ ВЫПОЛНЕНО
**Проблема:** По 2 запроса на каждый проект при генерации отчёта.

**Решение:** 
- 2 REST API запроса по всем проектам вместо 2×N проектов
- Прямой запрос через `requests` с полем `creator` (Jira Python client не возвращает creator)
- Обработка всех задач за 1 проход
- Агрегация статистики через словарь `project_stats`

**Реализация:**
- `fetch_issues_via_rest()` — получение задач через REST API
- `project_stats` — агрегация в памяти без повторных запросов
- `creator` доступен из `fields.creator` в REST API ответе

# 📊 Таблицы отчётов Jira Report System

Краткая справка по блокам отчёта.

## Обзор блоков

| Блок      | Стандартный режим      | Extra verbose режим         |
|-----------|------------------------|-----------------------------|
| summary   | 7 колонок              | 8 колонок (+ID проекта)     |
| assignees | 7 колонок              | 8 колонок (+ID исполнителя) |
| detail    | 10 колонок             | 11 колонок (+ID задачи)     |
| issues    | 8 колонок              | 9 колонок (+ID задачи)      |
| internal  | 10 колонок             | 11 колонок (+ID задачи)     |
| risk_zone | 7 колонок              | 8 колонок (+ID задачи)      |

---

## Формат данных в Extra verbose

| Поле          | Стандартный           | Extra verbose                    | Tooltip                  |
|---------------|-----------------------|----------------------------------|--------------------------|
| Проект        | WEB                   | WEB [10001]                      | project = '10001'        |
| Исполнитель   | Иванов Иван           | Иванов Иван [account-123]        | assignee = 'account-123' |
| Статус        | Done (done)           | Done (done) [5]                  | status = '5'             |
| Тип задачи    | Bug                   | Bug [1]                          | issuetype = '1'          |
| Дата          | 2026-03-10            | 2026-03-10 [duedate]             | duedate                  |
| Факт (ч)      | 2.5                   | 2.5 [timespent]                  | timespent                |
| Автор         | Петров Петр           | Петров Петр [account-456]        | creator = 'account-456'  |

---

## Важное замечание

**Заголовки колонок не меняются** в extra_verbose режиме.  
Добавляется только отдельная колонка **ID** после **URL**.

**Пример для detail:**
- Стандартный:     `['URL', 'Дата решения', 'Дата исполнения', ...]`
- Extra verbose:   `['URL', 'ID', 'Дата решения', 'Дата исполнения', ...]`

---

## Примеры использования

### Консольный режим
```bash
# Стандартный режим
python3 jira_report.py -b summary,detail

# Extra verbose режим
python3 jira_report.py -b summary,detail -vv
```

### Web-интерфейс
Чекбокс "Показывать ID объектов (extra verbose)" в форме отчёта.

### API
```json
{
  "projects": ["WEB"],
  "blocks": ["summary", "detail"],
  "extra_verbose": true
}
```

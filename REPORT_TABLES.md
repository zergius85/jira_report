## Сводка по проектам (summary)

Агрегированные данные по каждому проекту.

### Стандартный режим

| Колонка | Пример |
|---------|--------|
| Клиент (Проект) | `WEB` |
| Задач закрыто | `15` |
| Корректных | `12` |
| С ошибками | `3` |
| Оценка (ч) | `45.5` |
| Факт (ч) | `42.0` |
| Отклонение | `3.5` |

### Extra verbose режим

| Колонка | Пример |
|---------|--------|
| Клиент (Проект) | `WEB [10001]` |
| **ID** | `10001` |
| Задач закрыто | `15` |
| Корректных | `12` |
| С ошибками | `3` |
| Оценка (ч) | `45.5` |
| Факт (ч) | `42.0` |
| Отклонение | `3.5` |

**Tooltip:** `project = '10001'`

## Детализация по задачам (detail)

Полные данные по каждой закрытой задаче.

### Стандартный режим (10 колонок)

| Колонка | Пример |
|---------|--------|
| URL | `https://jira/browse/WEB-123` |
| Дата решения | `2026-03-10` |
| Дата исполнения | `2026-03-10` |
| Дата создания | `2026-03-01` |
| Проект | `WEB` |
| Статус | `Done (done)` |
| Задача | `Исправить баг` |
| Исполнитель | `Иванов Иван` |
| Факт (ч) | `2.5` |
| Тип | `Bug` |

### Extra verbose режим (11 колонок)

| Колонка | Пример |
|---------|--------|
| URL | `https://jira/browse/WEB-123` |
| **ID** | `10500` |
| Дата решения | `2026-03-10 [resolutiondate]` |
| Дата исполнения | `2026-03-10 [duedate]` |
| Дата создания | `2026-03-01 [created]` |
| Проект | `WEB [10001]` |
| Статус | `Done (done) [5]` |
| Задача | `Исправить баг` |
| Исполнитель | `Иванов Иван [account-123]` |
| Факт (ч) | `2.5 [timespent]` |
| Тип | `Bug [1]` |

**Tooltip:** `project = '10001'`, `status = '5'`, `assignee = 'account-123'`

## Сравнение режимов

### Формат данных

| Поле | Стандартный | Extra verbose | Tooltip |
|------|-------------|---------------|---------|
| Проект | `WEB` | `WEB [10001]` | `project = '10001'` |
| Исполнитель | `Иванов Иван` | `Иванов Иван [account-123]` | `assignee = 'account-123'` |
| Статус | `Done (done)` | `Done (done) [5]` | `status = '5'` |
| Тип задачи | `Bug` | `Bug [1]` | `issuetype = '1'` |
| Дата | `2026-03-10` | `2026-03-10 [duedate]` | `duedate` |
| Факт (ч) | `2.5` | `2.5 [timespent]` | `timespent` |

### Количество колонок

| Блок | Стандартный | Extra verbose | Разница |
|------|-------------|---------------|---------|
| summary | 7 | 8 | +1 (ID) |
| assignees | 7 | 8 | +1 (ID) |
| detail | 10 | 11 | +1 (ID) |
| issues | 8 | 9 | +1 (ID) |
| internal | 10 | 11 | +1 (ID) |
| risk_zone | 7 | 8 | +1 (ID) |

### Важное замечание

**Заголовки колонок не меняются** в extra_verbose режиме. Добавляется только отдельная колонка `ID` после `URL`.

## Примеры использования

### Консольный режим
```bash
# Стандартный режим
python3 jira_report.py -b summary,detail

# Extra verbose режим
python3 jira_report.py -b summary,detail -vv
```

### Web-интерфейс
Чекбокс "Показывать ID объектов (extra verbose)" в форме отчёта.

### API
```json
{
  "projects": ["WEB"],
  "blocks": ["summary", "detail"],
  "extra_verbose": true
}
```
